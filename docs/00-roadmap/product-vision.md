# Product Vision

## 1. 项目定位

LifeLab 是一个面向学习的原生 Android 综合案例 app。它的首要目标不是商业化，而是作为一个接近正式团队标准的示例工程，帮助开发者系统掌握 Kotlin、Android 原生开发与现代工程实践。

## 2. 核心目标

- 用一个统一业务壳承载多个典型 app 场景，而不是堆叠多个互不相干的小 demo
- 以正式项目的结构、边界和质量标准作为实现基线
- 用 Kotlin 现代语法与 Jetpack 体系组织应用开发
- 在可控复杂度内引入 Clean Architecture 和务实 DDD

## 3. 目标用户

- 有 Java 背景，准备进入 Kotlin/Android 原生开发的开发者
- 希望通过一个完整工程学习 Android 项目分层、状态流、测试和常见业务场景的开发者
- 需要一个能持续扩展的个人样板工程的开发者

## 4. 成功标准

项目达到以下标准时，视为第一阶段成功：

- 能完整演示首页、任务、习惯、发现、搜索、消息、我的等核心模块
- 所有页面都按统一状态模型管理加载态、内容态、空态、错误态
- 网络、本地存储、导航、依赖注入、测试基线都已落地
- 项目结构足够清晰，新增功能不需要推翻现有边界
- Kotlin 初学者可以通过该项目看到高频语法在真实场景中的使用方式

## 5. 非目标

以下内容不作为第一阶段目标：

- 追求完整商业闭环
- 引入过重的 DDD 战术模式
- 一开始就拆成大量 Gradle feature modules
- 兼容过多平台或历史 Android 技术债

## 6. 产品原则

- 一个产品壳，多个典型业务场景
- 一个功能一个清晰边界
- 一个页面一套统一状态模型
- 一个需求优先最小可维护实现，而不是炫技实现
